bash
ng new parking-xon
cd parking-xon

Luego crea tu componente principal:

bash
ng generate component AndresDiaz

Esto genera los archivos:

bash

  ├── andres-diaz.component.css
  ├── andres-diaz.component.ts
  └── andres-diaz.component.spec.ts

Y Angular ya lo declara automáticamente en tu app.module.ts.

🌐 2. Configura el enrutamiento

Queremos una estructura de rutas para futuras páginas como:

  • /buscar

  • /reservar

  • /pagar

Edita el archivo app-routing.module.ts:

ts
import { NgModule } from '@angular/core';
import { RouterModule, Routes } from '@angular/router';
import { AndresDiazComponent } from './andres-diaz/andres-diaz.component';

const routes: Routes = [
  { path: '', component: AndresDiazComponent },
  // Aquí agregarás otras rutas luego
  { path: '**', redirectTo: '' } // Redirige rutas no válidas al home
];

@NgModule({
  imports: [RouterModule.forRoot(routes)],
  exports: [RouterModule]
})
export class AppRoutingModule {}
